## Landmines

- **Gates are now CI-held and permanent** — before any styling change, know that
  `tests/design/greppable-gates.test.ts` will bite. Its exemption maps are the
  canonical documentation: inline-style per-file ceilings (canvas 28, agent-shell 23,
  components 13, editor 9, sidebar 8, ghosts 3 — ceilings, so a new static style in an
  exempted file only fails past the ceiling; keep them tight when counts drop), hex
  allowlist (above), transition vocabulary.
- **Transition vocabulary ratified as two tiers**: `--t-micro/fast/med/slow/surface/
  reveal/spring` + the full exact-value `--transition-*` family (hover, focus-ring,
  tooltip, modal-fade — gates-test found the tree uses all of these, broader than the
  pair slice 6 named). Layer 4 collapses them; don't unify earlier.
- **Muted-text contrast pin** (≥4.0, TODO-tagged in contrast.test.ts) is deliberate
  debt for the Layer 4 retune: darken surfaces or lift the muted step, then delete the
  pin's `< 4.5` twin assert.
- **Visual baselines are darwin-only** (`-darwin.png`). A Linux CI runner would need
  its own baseline set; nothing enforces cross-platform today.
- **Changing any token value or te- rule can fail `e2e/visual.spec.ts`** — that's the
  point; regenerate baselines deliberately with `--update-snapshots` and eyeball the
  diff, never blindly.
- **Gallery determinism contract**: no timers/network/IPC/dates/random in Gallery.tsx;
  the spinner is the only animated element and stays isolated under
  `data-testid="gallery-spinner"`. New primitives should gain a gallery section + a
  visual spec entry.
- **Vendored prelude** at the top of index.css: still untouchable; the hex-gate
  sentinel comment (`END vendored Tailwind prelude`) is load-bearing for the gates
  test — don't reword it.
- **Append-order dependencies and deliberate compound selectors** in index.css
  (slices 5–6) still hold: later sections win equal-specificity ties; don't reorder or
  simplify compounds.
- **`--t-spring` / `--t-reveal` are CSS-only** (Theme.tsx re-emits only
  micro/fast/med/slow/surface). Unchanged.
- **Imperative styling boundary**: `container.style.animation = 'te-scale-in 150ms
  ease-out'` (slash-command.tsx, wikilink-suggestion.tsx) and NoteCard's cssText are
  outside the transition gate by design (animation ≠ transition; imperative ≠
  declaration). Documented in the gates test.
- **e2e runs rewrite `e2e/fixtures/test-vault/.machina/state.json`** — `git restore`
  before every commit; never commit it.
- **npm audit still has 9 pre-existing vulnerabilities** (3 high, sharp/libvips) —
  needs its own deps pass, unrelated to this work.
- Cursor/GitLens intermittently hold `.git/index.lock` — retry a failed commit before
  touching the lock. eslint uses `--cache` (`--no-cache` if stale); npm installs need
  `--cache /tmp/npm-cache-te`. Skip-worktree check: `git ls-files -v | grep ^S`.
